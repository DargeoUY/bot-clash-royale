⚔️ Asistente Royale

Bot de Discord para administración de clanes de Clash Royale

Tracking de guerras · Inactividad · WhatsApp · Ranking diario · Puntos · Telegram

🚀 Cómo instalar
🤖 Invitar Bot de Discord 📱 Invitar Bot de Telegram

✨ Funcionalidades

🔄 Sync Automático

Importa miembros del clan cada 1 hora. Detecta altas, bajas y reingresos automáticamente.

⚠️ Detección de Inactividad

Umbrales configurables (2–14 días). Notificaciones automáticas al canal de alertas. Modo vacaciones.

📊 Ranking Diario

Victorias/Derrotas, Donaciones, Copas y Guerra de Clanes. Stats diarias con delta real (no totales históricos).

📱 Grupo de WhatsApp

Verifica que el jugador esté en el clan antes de compartir el link del grupo. Por servidor.

🏆 Ganadores Semanales

Cada lunes se anuncian los ganadores de cada categoría. Roles automáticos por 1 semana con insignia separada.

⭐ Sistema de Puntos

Puntos por guerra, actividad, donaciones. Bonus y penalizaciones manuales. Ranking semanal/mensual.

🏰 Multi-Servidor

Cada guild de Discord tiene su propio clan, canales, roles y configuraciones. Sin mezclar datos.

📲 Telegram

Notificaciones automáticas vía Telegram. Comandos desde el grupo. Bienvenida personalizada con imagen. Vinculación con Discord.

⚠️ Sistema de Inactividad

Umbrales configurables (default 2 días). La escala se ajusta según el tamaño del clan:

Tamaño del ClanAvisoInactivoExpulsión sugerida
43 – 502 días4 días6 días
30 – 422 días5 días10 días
< 302 días7 días14 días
🏖️ Modo Vacaciones: /ausencia <dias> pausa las notificaciones (máx 20 días/mes).

📱 Flujo del Grupo de WhatsApp

Líder configura el link:
/config link-whatsapp https://chat.whatsapp.com/...
Miembro abre el formulario:
/whatsapp → completa nombre en el juego y número de celular.
El bot verifica contra el clan:
Compara el nombre ingresado con los miembros del clan vía API.
Resultado:
✅ Coincide → guarda datos + muestra el link del grupo.
❌ No coincide → "No verificado, el nombre no está en el clan."

🏆 Ganadores Semanales

Cada lunes a las 10 AM UTC se calculan los ganadores acumulando los stats diarios de la semana:

CategoríaRol asignadoMétrica
⚔️ Victorias / Derrotas🏆 Campeón del MesMejor win rate semanal
💎 Donaciones de Cartas💎 Donador LegendarioMás cartas donadas
🏆 Mayor Cantidad de Copas🏆 Rey de CopasMás copas ganadas
⚔️ Guerra de Clanes⚔️ Guerrero ÉliteMás fama acumulada
🔝 Los roles son hoisted: aparecen separados del resto de miembros en la barra lateral.
🔄 Se quitan automáticamente al ganador anterior y se asignan al nuevo cada lunes.

🚀 Cómo Instalar en tu Servidor

Creá una API Key en Supercell:
Entrá a developer.clashroyale.com, creá una key y en "Whitelist IP" poné: 45.79.218.79
Creá un Bot en Discord:
Andá a Discord Developer Portal, creá una aplicación, andá a Bot y copiá el token.
Invitá el Bot a tu servidor:
Usá este link (reemplazá CLIENT_ID):
https://discord.com/oauth2/authorize?client_id=...
Cloná el repositorio y configurá: 
git clone https://github.com/DargeoUY/bot-clash-royale.git
cd bot-clash-royale
cp .env.example .env   # editar con tus tokens
docker compose up -d
En Discord:
/auto-setup #CLAN_TAG
Para Telegram (opcional):
Agregá el bot a tu grupo con el link que te da /auto-setup. El bot te dará un código.
Después: /vincular → completá el tag del clan y el código.
Configurar en el .env:
TELEGRAM_BOT_TOKEN= (token de @BotFather). Un solo bot para todos los clanes.
